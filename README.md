## Installation

**Clone the repo:**

```bash
git clone https://github.com/felipebrsk/gcstatus.git
```

**Enter the project folder:**

```bash
cd gcstatus
```

**Install dependencies:**

```bash
npm install
```

or

```bash
yarn install
```

**Run the app:**

```bash
npm start
```

or

```bash
yarn start
```

**Open your browser:**
